## Instalación

1. Clona y entra en el repositorio con el siguiente comando:

```bash
git clone https://github.com/GaelQuesadilla/potro-tetris.git
cd potro-tetris
```

2. Instala las dependencias ejecutando desde el directorio principal el siguiente comando:

```bash
pip install -r requirements.txt
```
